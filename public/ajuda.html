Tutorial FiscoBot Pro

Guia completo — do zero ao avançado em minutos.

🏗 Como o FiscoBot funciona

Antes de usar, entenda a arquitetura em 30 segundos:

# Quando você clica no atalho:

electron.js    → abre a janela nativa do Windows
    ↓
server.js      → sobe um servidor Express na porta 3737
    ↓
Janela Electron → carrega http://localhost:3737
    ↓
Você           → interage com o painel visual
    ↓
Servidor       → executa comandos (SEFAZ, macros, certs...)

O servidor nunca fica visível — roda em segundo plano. A janela é apenas o painel de controle. Tudo se comunica via HTTP e WebSocket internamente.

ℹ️
Por que Electron? Permite empacotar tudo em um único instalador .exe — sem precisar instalar Node.js, sem abrir terminal, sem localhost visível para o usuário.

1 Instalação

Pré-requisito: Node.js 18+ instalado no Windows.

1
Extraia o ZIP em uma pasta permanente (ex: C:\FiscoBot\)
2
Dê duplo clique em INSTALAR.bat
3
Aguarde a instalação das dependências (1–3 minutos, baixa ~120MB)
4
O app abre automaticamente e cria um atalho na Área de Trabalho

Estrutura de arquivos

fiscobot-electron\
├── INSTALAR.bat       ← execute isso PRIMEIRO
├── rodar.bat          ← para abrir depois (sem janela)
├── rodar-debug.bat    ← se der erro, use este (mostra log)
├── DESINSTALAR.bat    ← para remover o app
├── electron.js        ← núcleo Electron (não abra)
├── server.js          ← servidor Express (não abra)
├── preload.js         ← ponte segura renderer↔main
└── public\
    ├── index.html     ← interface principal
    ├── app.js         ← lógica do painel
    ├── conferencia.html ← ferramenta NF-e × SEFAZ
    └── ajuda.html     ← você está aqui
⚠️
Não mova a pasta! Os atalhos apontam para o caminho fixo. Se mover, execute o INSTALAR.bat novamente para recriar os atalhos.

3 🔏 Certificados Digitais

O certificado digital é necessário para a consulta SEFAZ real e para acessar portais que exigem autenticação. O FiscoBot suporta certificados A1 (arquivo .pfx ou .p12).

1
Localize seu arquivo .pfx ou .p12 no computador
2
Arraste o arquivo para a área tracejada da sidebar esquerda — ou clique na área para abrir o explorador
3
Digite a senha do certificado no campo abaixo da área de drop
4
Clique em IMPORTAR — o certificado aparece na lista com badge verde ✓

🔒 Segurança: o arquivo fica salvo em %APPDATA%\.fiscobot\certs\ com permissão restrita. A senha é criptografada com AES-256-GCM e nunca fica em texto puro no disco.

Gerenciar certificados

Na lista de certificados ativos, cada item tem dois botões:

BotãoAção
⬇Baixa o arquivo do certificado
✕Remove o certificado do FiscoBot (não apaga o original)

Clique em um certificado para selecioná-lo (borda dourada) e depois em Exportar / Baixar.

5 ⚙ Macros

Macro = sequência de cliques e digitações que o app repete automaticamente. Ideal para tarefas repetitivas como preencher filtros, gerar relatórios ou navegar por menus complexos.

Gravar um macro

1
Clique em ☰ no header para abrir o painel lateral
2
Vá na aba ⚙ Macros
3
Clique em ⏺ Gravar — a barra vermelha indica que está gravando
4
Execute as ações desejadas no painel (cliques em botões, preenchimento de campos...)
5
Clique em ⏹ Parar
6
Digite um nome para o macro e clique em 💾
⏺ Gravando — 3 passos [⏺ Gravar] [⏹ Parar] [600 ms ↕] ────────────────────────── ⚙ Login eCAC 3 passos · 600ms · 15/03/2025 [▶ Rodar] [✕] 📑 Gerar DARF 8 passos · 800ms · 10/03/2025 [▶ Rodar] [✕]

Executar um macro

Clique em ▶ Rodar ao lado do macro salvo. A barra de progresso mostra cada passo sendo executado.

Campo de delay (ms)

Controla a velocidade entre cada passo. Valores recomendados:

DelayQuando usar
300msAções simples em páginas rápidas
600msPadrão — funciona na maioria dos portais
1200msPortais lentos ou com validação assíncrona

6 📝 AutoFill

Salva os dados das suas empresas e preenche formulários automaticamente, detectando campos por nome, ID e placeholder.

Criar um perfil de empresa

1
Clique em ☰ → 📝 Fill
2
Preencha os campos: CNPJ, Razão Social, IE, IM, e-mail, telefone, endereço, CEP, UF
3
Clique em 💾 para salvar o perfil
4
Para usar em outro perfil: clique em + para limpar e criar um novo

Preencher uma página

1
Selecione o perfil desejado no menu suspenso
2
Clique em ⚡ Preencher Página
3
Os campos compatíveis ficam com borda dourada por 600ms confirmando o preenchimento
💡
O AutoFill detecta campos por palavras-chave: cnpj, razao, email, telefone, cep, etc. Funciona em português com variações de grafia.

8 🧠 Agente IA — Léo

O Léo é um assistente especialista em tributação brasileira integrado ao FiscoBot. Responde dúvidas sobre Simples Nacional, ICMS, NF-e, eSocial, SPED e muito mais — com tom descontraído e direto.

Configuração inicial (obrigatória)

1
Acesse console.anthropic.com e crie uma conta
2
Vá em API Keys → Create Key e copie a chave (começa com sk-ant-)
3
No FiscoBot: ☰ → 🧠 IA
4
Cole a chave no campo e clique em ✓ — a bolinha fica verde quando a key é válida

Usando o Léo

Use os chips de atalho (DAS, DIFAL, IRPJ LP...) para perguntas frequentes, ou digite livremente no campo de texto. Pressione Enter para enviar ou Shift+Enter para quebrar linha.

💡 Exemplos de perguntas:
"Como calcular o DAS de uma empresa no Anexo III com receita de R$ 80.000?"
"Qual o CFOP correto para remessa de bonificação dentro do estado?"
"Posso creditar PIS/COFINS sobre energia elétrica no regime não-cumulativo?"

Modelos disponíveis (Configurações → Agente IA)

ModeloCaracterísticas

Opus 4.6 🧠Respostas mais elaboradas para casos complexos
Haiku 4.5 🚀Mais rápido para perguntas simples

9 ⚖️ Conferência NF-e × SEFAZ

Compara as notas do seu sistema (Domínio, TOTVS, etc.) com a situação real na SEFAZ, detectando automaticamente divergências.

Como usar

1
Clique no ícone ⚖️ no header
2
Exporte um relatório CSV do seu sistema com as colunas: chave, número, empresa, cnpj, valor, data, status
3
Arraste o arquivo para a tela ou clique para selecionar
4
Clique em 🔍 Consultar SEFAZ — o app consulta cada nota usando seu certificado A1
5
Analise o relatório e exporte as divergências em CSV

Níveis de divergência

NívelSignificadoAção sugerida
🔴 CríticoNota ativa no sistema mas cancelada/denegada na SEFAZRegularização imediata — risco fiscal alto
🟠 AtençãoNota não encontrada na SEFAZVerificar chave de acesso e reemissão
🔵 RevisarErro na consulta ou status divergenteConsultar manualmente no portal
✅ OKStatus confere entre sistema e SEFAZNenhuma ação necessária
⚠️
Certificado obrigatório: a consulta real à SEFAZ exige um certificado A1 importado na sidebar. Sem certificado, a consulta falha com erro "Sem certificado".

🔧 Problemas Comuns

O app não abre ao clicar no atalho

Execute rodar-debug.bat — ele mostra o terminal com a mensagem de erro exata. Geralmente é ausência do Node.js ou node_modules corrompido.

Mensagem "Porta 3737 já em uso"

O FiscoBot já está rodando em segundo plano. Procure o ícone ⚡ na bandeja do sistema (canto inferior direito do Windows) e dê duplo clique para abrir a janela.

Certificado não é aceito (erro ao importar)

Verifique: (1) o arquivo é .pfx ou .p12; (2) a senha está correta; (3) o certificado não expirou. Tente abrir o .pfx no Windows para confirmar a senha.

Léo não responde / erro no chat

A bolinha ao lado do campo de API Key deve estar verde. Se estiver cinza, a key não foi salva. Se estiver vermelha, a key é inválida — verifique se começa com sk-ant-.

Consulta SEFAZ retorna "Sem certificado"

Importe um certificado A1 (.pfx) na sidebar esquerda antes de usar a Conferência NF-e. A consulta SEFAZ exige autenticação mútua TLS.

Macro não executa os passos corretamente

Aumente o delay (ex: de 600ms para 1200ms) em Configurações → Macros. Portais governamentais costumam ter validações lentas que precisam de mais tempo entre os passos.

📞
Ainda com problema? Execute rodar-debug.bat, copie o erro que aparecer no terminal e consulte o Léo no app — ele conhece a arquitetura e pode ajudar a diagnosticar.
